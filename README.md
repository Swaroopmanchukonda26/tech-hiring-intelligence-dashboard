## 📌 Project Overview
An end-to-end data analytics project analyzing tech 
industry layoffs, fresher placement trends and HR 
employee attrition patterns using Python, SQL, 
Power BI and Streamlit.

## 📊 Datasets Analyzed
| Dataset | Records | Source |
|---------|---------|--------|
| Tech Layoffs | 4,428 records | Kaggle |
| Student Placement | 1,00,000 records | Kaggle |
| HR Attrition | 1,470 records | IBM/Kaggle |
| **Total** | **1,05,898 records** | |

## 🔑 Key Insights Found
- 🔴 Amazon had highest layoffs — 58,124 employees
- 🔴 2023 was worst year for tech layoffs globally
- 🟡 Overall fresher placement rate — 68.5%
- 🟡 Students with 2+ internships — 73% placement rate
- 🟡 Average placed student salary — 17.31 LPA
- 🟢 Overall HR attrition rate — 16.1%
- 🟢 Higher job level = significantly higher salary
